# lovemeter

Partner kriterleri ile iliski degerlendirme uygulamasi.

## Kurulum

```bash
pip install -r requirements.txt
```

## Calistirma

```bash
streamlit run streamlit_app.py --server.port 8511
```

Not:
- Eger 8511 doluysa farkli bir port kullanin (orn: 8502).
- Uygulamanin giris dosyasi `streamlit_app.py`, ana arayuz kodu `iliskidegerledirmeformu.py` icindedir.
